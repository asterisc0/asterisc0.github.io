Comandos Termux

Comandos Básicos

  • pkg update && pkg upgrade - Invocar el ritual de actualización
  • pkg install nombre_paquete - Convocar e instalar un paquete
  • termux-setup-storage - Configurar el almacenamiento místico
  • cd nombre_directorio - Entrar al reino del directorio
  • ls - Listar los elementos del reino
  • cp archivo destino - Duplicar el pergamino
  • mv archivo destino - Mover o renombrar el pergamino
  • rm archivo - Borrar el pergamino maldito
  • mkdir nombre_directorio - Crear un portal dimensional
  • rmdir nombre_directorio - Cerrar el portal dimensional vacío
  • cat archivo - Leer las runas del archivo
  • echo "texto" > archivo - Conjurar un archivo con texto
  • echo $variable - Revelar el valor de una variable mística
  • chmod permisos archivo - Cambiar los encantamientos del archivo
  • chown usuario:grupo archivo - Cambiar el propietario del archivo por un hechicero
  • pwd - Conocer el camino actual en el inframundo
  • whoami - Invocar al ser interior
  • date - Descifrar la fecha y hora en las estrellas
  • clear - Limpiar la bola de cristal
