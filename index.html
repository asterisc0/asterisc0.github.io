Comandos Termux

Comandos Básicos

  • pkg update && pkg upgrade - Actualizar paquetes
  • pkg install nombre_paquete - Instalar un paquete
  • termux-setup-storage - Configurar almacenamiento
  • cd nombre_directorio - Cambiar de directorio
  • ls - Listar archivos en el directorio
  • cp archivo destino - Copiar un archivo
  • mv archivo destino - Mover o renombrar un archivo
  • rm archivo - Eliminar un archivo
  • mkdir nombre_directorio - Crear un directorio
  • rmdir nombre_directorio - Eliminar un directorio vacío
  • cat archivo - Mostrar el contenido de un archivo
  • echo "texto" > archivo - Crear un archivo con texto
  • echo $variable - Mostrar el valor de una variable
  • chmod permisos archivo - Cambiar permisos de un archivo
  • chown usuario:grupo archivo - Cambiar propietario de un archivo
  • pwd - Mostrar el directorio actual
  • whoami - Mostrar el nombre de usuario actual
  • date - Mostrar la fecha y hora actuales
  • clear - Limpiar la pantalla
